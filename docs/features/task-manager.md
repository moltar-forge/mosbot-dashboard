# Task Manager (Overview)

The **Task Manager Overview** is the runtime operations view of MosBot OS.

## Purpose

- Real-time monitoring of **agent sessions** and **task metrics**
- Quick health check: “is anything running, stuck, or idle?”

## What you’ll see

- Session KPIs (example): **Running / Active / Idle / Total**
- Usage metrics (example): **Tokens Used** and **Total Cost** (derived from session data)
- Session lists (example):
  - **Active Sessions** (running + active)
  - **Idle Sessions**
- **Cron Jobs**: scheduled automation configuration and visibility

## Update behavior

The dashboard refreshes operational data periodically and when the browser tab becomes visible again.
